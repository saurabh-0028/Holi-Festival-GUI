## ✨ Features

- 🎆 **Particle System** — Mouse ke saath rang barsate hain
- 🌈 **Animated HOLI Title** — Rainbow gradient + neon glow
- 💥 **Color Blobs** — Click karo, burst ho jata hai!
- 🎉 **"Throw Colors!" Button** — Mega color explosion
- 🪷 **Spinning Rangoli SVG** — 12 petals, fully animated
- 🖱️ **Custom Glowing Cursor** — Neon color trail
- 👑 **Premium Signature** — Mr. Selfish branding at bottom
- 📱 **Responsive Design** — Mobile friendly

## 🎮 How to Use

1. `holi_mrselfish.html` file download karo
2. Browser mein open karo (Chrome/Firefox recommended)
3. **Mouse ghuma** — color trail dekho
4. **Color blobs click karo** — burst effect!
5. **"Throw Colors!" button dabaao** — mega explosion!
